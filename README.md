# 整合包介绍

“还有很多故事等待讲述！”

Tales of Glarthford是一张MC原版的经典冒险地图，以全角色配音、打破边界玩法、回合制战斗和一系列故事线为特点。

__✧ 10+独特的故事__

Tales of Glarthford分为10个独立的故事（Tales）线，发生于格拉斯福德（Glarthford）小镇及其周边区域。每个故事会为格拉斯福德的世界引入新的情景，你需要找到新的方法解决特定问题。每个故事都有若干种解法——你会选择哪一种呢？此外，还有一个隐藏的，第十一个故事，你能找到吗？

__✧ 190+带有配音的角色__

地图的特点之一便是超过190个的全配音可互动角色。你不止可以和他们交谈与贸易，还有更多可互动的形式，例如趁他们睡着的时候偷取他们的物品。此外，地图的昼夜交替也会影响角色的发言与行为。

__✧ 回合制战斗__

此地图采用独特的回合制战斗机制，为求在动作冒险与策略规划之间找到平衡。与最多两名玩家组队，再带上游戏内的部分角色，甚至可能是宠物，一起面对一系列前所未见的Boss与其它强大的敌人。留意你在主世界中收集的各种物品，并在战斗中合理利用。你能成功击败所有的Boss吗？只有一种方法可以确认。

__✧ 自定义音乐__

地图还带有自定义音乐，内含许多来自BartTheBart、TTEXTT、Green Greeny、Enshano等更多创作者编写的曲目，以提升玩家在格拉斯福德世界的沉浸感！

__✧ 探索__

格拉斯福德的世界充满了各种细节与秘密！隐藏角色、奇怪的生物或者隐秘地点——总有新事物等着你去发现！
